# Figure autofix modes\n\n`autofix_figure_table_refs.py` supports three citation output modes:\n\n- `support-note`\n  - Draft-oriented mode\n  - Produces prose like “相关判断可结合 xxx 进一步支撑”\n\n- `inline-marker`\n  - Pre-final mode\n  - Produces visible markers like `[2]` / `(Lee, 2024)` placeholders from plan hints\n\n- `internal-anchor`\n  - Unified pipeline mode\n  - Produces internal anchors like `[CITE:baseline comparison|lee2024benchmark]`\n  - Best choice when the draft will later pass through `render_final_citations.py`\n\n## When to use which\n\n- Early drafting → `support-note`\n- Review / semi-final draft → `inline-marker`\n- Full automated final rendering pipeline → `internal-anchor`\n\n## Rule of thumb\n\nIf the draft will enter the unified citation rendering chain, prefer `internal-anchor`.\n
