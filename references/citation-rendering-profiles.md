# Citation rendering profiles\n\n`render_final_citations.py` can accept `--citation-profile-json` to apply template-aware rendering preferences.\n\n## Supported profiles\n\n- `cn-thesis-brackets`\n  - GB/T 7714 numeric bracket style\n  - Chinese punctuation cleanup\n\n- `cn-thesis-superscript`\n  - Reserved for superscript-like numeric citation rendering\n  - Current support is partial / formatting-oriented\n\n- `apa-inline`\n  - APA author-year style\n  - Chinese-context polish can convert terminal `(Lee, 2024)` into `（Lee, 2024）`\n\n## Inputs\n\nA profile JSON may contain:\n\n- `style`\n- `profile`\n- `bracketStyle`\n- `zhPunctuationMode`\n- `apaParenModeZh`\n\n## Recommendation\n\nPrefer passing the citation rendering block selected from the resolved layout template so the final rendering layer follows the same template assumptions as the paper layout layer.\n
